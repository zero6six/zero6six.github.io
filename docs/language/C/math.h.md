
math.h 中定义的函数接收值和返回值都为 double，不是时应该会强制转换。

注意：XDOJ 中，math.h 的输入值必须为 double，否则要进行显式类型转换，不然编译错误。~~垃圾 XDOJ。~~

- log(x) 返回自然对数
- log10(x) 返回常用对数
- exp(x) 返回 e 的 x 次幂
- pow(x, y) 返回 x 的 y 次幂
- sqrt(x) 返回 x 的平方根
- floor(x) 返回 x 向下取整
- ceil(x) 返回 x 向上取整
- fabs(x) 返回绝对值
