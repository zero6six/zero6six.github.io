```c
for (int i=0; a[i]!='\0'; i++){
for (int i=0; i<strlen(a); i++){ // 需要导入 string.h
```

字符串在声明和赋值时可以使用 `char title[]="C Programming"`，但是 `char title[50];title="C Programming";` 是非法的，这是因为数组名在 C 语言中表示数组的首地址，不能被赋值。
